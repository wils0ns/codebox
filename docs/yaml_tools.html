codebox: YAML utilities¶

YAML utility tools

codebox.yaml_tools.load(path, recursive=False, match_pattern=None, ignore_empty=False, parse_jinja=False, jinja_context=None)[source]¶

Loads and parses a file or a folder containing yaml files. All existing dictorinaries will be deeply merged.

Parameters

  • path (str) – File or directory path.

  • recursive (bool, optional) – Defaults to False. Whether or not to include subdirectoties.

  • match_pattern (str, optional) – A regular expression to be used to filter the files to be load based on the file’s full name.

  • ignore_empty (bool, optional) – Whether or not to ignore empty files.

  • parse_jinja (bool, optional) – Whether or not to parse Jinja code.

  • jinja_context (dict, optional) – The Jinja Context.

Return type

dict

Returns

dict
