codebox.dict_tools.merge(*dicts)[source]¶

Deep merges the right most dictionaries into the left most one.

Parameters

*dicts (dict) – dictionaries to be merged.

Examples

>>> a = {'a': 1, 'b': 2}
>>> b = {'b': 3, 'c': {'c1': 4, 'c2': 3}}
>>> c = {'a': 3, 'c': {'c1': 3 }}
>>> print(merge(a,b,c))
{'c': {'c1': 3, 'c2': 3}, 'a': 3, 'b': 3}
Return type

dict

codebox.dict_tools.update_nested(the_dict, nested_key, value)[source]¶

Updates nested keys inside a dictionary

Parameters

  • the_dict (dict) – The dictionary to be updated

  • nested_key (str) – The nested key

  • value (Any) – The value to be set to the nested key

Examples

>>> t = {}
>>> update_nested(t, 'a:b:c', 1)
>>> t
{'a': {'b': {'c': 1}}}
>>> update_nested(t, 'a:b:d', 1)
>>> t
{'a': {'b': {'c': 1, 'd': 1}}}
>>> update_nested(t, 'a:b:c', 2)
>>> t
{'a': {'b': {'c': 2, 'd': 1}}}
